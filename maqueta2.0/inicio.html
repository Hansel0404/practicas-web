es el proceso de crear instrucciones para que una computadora o software ejecute una tarea específica. Estas instrucciones se escriben en un lenguaje de programación y se conocen como código.

La mayoría de las personas alrededor del mundo utilizan una computadora para realizar tareas específicas ya sea para trabajar, entretenerse y aprender, entre muchas otras actividades cotidianas. Las órdenes que se dan al dispositivo se ejecutan gracias a un conjunto de recursos del sistema operativo conocido como software. El desarrollador de software es la mente creativa encargada de diseñar y optimizar estos recursos.
